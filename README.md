# 💠 Sentiric Workflow Service

Olay güdümlü (Event-Driven) State Machine orkestratörüdür. Çağrı yaşam döngüsünü (SAGA) yönetir.

## 🚀 Hızlı Başlangıç
```bash
go run cmd/workflow-service/main.go
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/services/workflow.spec.yaml](https://github.com/sentiric/sentiric-spec)
